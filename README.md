# CogNet: a Large-Scale Cognate Database

![alt text](http://ukc.disi.unitn.it/wp-content/uploads/2019/05/Cognate_fish.jpg)

CogNet is a large-scale database of cognate pairs: it contains 5.9 million cognates in 338 languages and 38 writing systems. It was automatically constructed from wordnets and dictionaries contained within the UKC resource, as described in our paper.

## CogNet Structure
Please have a look at the following website.
